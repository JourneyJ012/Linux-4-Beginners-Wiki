## Dispelling Linux Rumors

### "Linux requires terminal knowledge to function"
Most versions of Linux nowadays **come with** Firefox, a file explorer, a fully usable and customisable desktop, and everything else you need to hit the ground running.
### "Installing any software is difficult on Linux"
Linux distributions feature a software store that has the vast majority of things you will need. Most other software is released in a package that can be installed through a graphical installer. Only *some* is relatively complex, and they typically come with installation guides.
### "Linux is unstable"
Linux is as stable as you make it. If it was unstable, NASA, Amazon, and thousands of other companies wouldn't use it.
### "Linux is harder to install than Windows"
Linux, when installed fresh, is easier to set up than Windows, which requires you to sign up to their services. Lenovo even allow you to buy some laptops with Linux installed, and it's cheaper than Windows!
