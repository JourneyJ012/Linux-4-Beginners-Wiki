Firefox is a web browser based on an entirely different codebase that the Chromium set of browsers (Edge, Chrome, Brave).

## Installation
### Linux Mint
Firefox is pre-installed on Mint. If you have uninstalled it, open the Software Manager, and search "Firefox" in the top left. It is recommended to get the one **not** marked as "`Flathub`".
